# Portafolio de Cristhofer Pérez Navarro

Este es mi portafolio personal, desarrollado con HTML y CSS. El propósito de este portafolio es mostrar mis habilidades y proyectos como programador, además de servir como punto de contacto.

## Tecnologías utilizadas

- **HTML**: Estructura y contenido del portafolio.
- **CSS**: Diseño y estilo del portafolio.
- **Font Awesome**: Para los íconos sociales de LinkedIn y GitHub.
  
## Características

- **Sección 'Sobre mí'**: Breve descripción de mi formación en Desarrollo de Software y Tecnologías de la Información.
- **Sección de Proyectos**: Muestra los proyectos en los que he trabajado y los enlaces a los repositorios.
- **Sección de Certificaciones**: Presenta mis certificaciones obtenidas en el área de desarrollo.
- **Íconos de redes sociales**: Incluye íconos de LinkedIn y GitHub para facilitar el contacto.
